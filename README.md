# NextJS Store

A NextJS based store with Mongo

## Installation

Clone the repo and use the package manager [npm](https://www.npmjs.com/) to install NextJS store.

```bash
npm install
npm run build

```

## Usage

```bash
npm run start
```

Point your browser to localhost:3000
